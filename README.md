#  Doctor Appointment System

##  Description

CLI-based Doctor Appointment System built using Python and MySQL. It allows users to manage patients, view doctors, and book appointments efficiently.

---

##  Features

* Register Patients
* Search Patients
* View Doctors
* Book Appointments
* Check Doctor Availability

---

##  Tech Stack

* Python
* MySQL

---

##  How to Run

1. Install dependencies:
   pip install mysql-connector-python

2. Run the project:
   python main.py
